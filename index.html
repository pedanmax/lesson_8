Псевдоклассы положения в коде

Обращение к первому элементу в блоке

:first-child { свойство:значение;}

:first-of-type { свойство:значение;}

Обращение к последнему элементу в блоке

:last-child { свойство:значение;}

:last-of-type { свойство:значение;}

Обращение к конкретным элементам в блоке

:nth-child(1) { свойство:значение;}

:nth-child(2) { свойство:значение;}

:nth-of-type(1) { свойство:значение;}

:nth-of-type(2) { свойство:значение;}

Псевдоэлементы :first-line и :first-letter

Псевдоэлемент :first-line задает стиль первой строки текста. Псевдоэлемент :first-letter определяет стиль первого символа в тексте элемента

Король и Королева псевдоэлементов :before и :after

:before применяется для отображения желаемого контента ДО элемента, к которому он добавляется.

:after используется для вывода желаемого контента ПОСЛЕ элемента, к которому он добавляется.
